# 開発サーバーの起動

このワークフローは、アルゴリズム解説アプリの開発サーバーを起動します。

## 手順

// turbo

1. プロジェクトディレクトリに移動

```bash
cd c:\Program2\T\pro_01\2026-02\2026-02-02_algorithm_cards
```

// turbo
2. 開発サーバーを起動

```bash
npm run dev
```

1. ブラウザで以下の URL にアクセス

```
http://localhost:5173/pro_01/
```

## 注意事項

- ポート 5173 が既に使用されている場合、Vite は自動的に別のポートを使用します
- ファイルを編集すると、ブラウザが自動的にリロードされます（Hot Module Replacement）
- サーバーを停止するには、ターミナルで `Ctrl+C` を押してください
