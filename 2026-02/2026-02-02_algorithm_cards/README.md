# アルゴリズムの解説

インタラクティブなビジュアライゼーションで、アルゴリズムの動作を直感的に理解できる教育用Webアプリケーションです。

![Algorithm Visualization](https://img.shields.io/badge/Status-Active-success)
![License](https://img.shields.io/badge/License-MIT-blue)

## 🎯 概要

このアプリケーションは、トランプカードを使ったビジュアライゼーションにより、以下のアルゴリズムの動作を**ステップバイステップ**で学習できます：

- **探索アルゴリズム**: 線形探索、二分探索
- **ソートアルゴリズム**: バブルソート、選択ソート、挿入ソート
- **その他**: 最大・最小値の探索

各ステップで**コードの該当行が強調表示**され、**変数の状態**がリアルタイムで表示されるため、アルゴリズムの思考プロセスを視覚的に追うことができます。

## ✨ 主要機能

### 📊 ビジュアライゼーション

- トランプカードによる直感的な表現
- ステップごとのアニメーション
- 比較・交換・移動の視覚的フィードバック

### 💻 コード表示

- **Macro** と **Python** の2言語対応
- 実行中のコード行を自動強調
- ステップIDベースの正確な同期

### 🎮 インタラクティブ操作

- **再生/一時停止**: 自動実行の制御
- **ステップ送り/戻し**: 1ステップずつ確認
- **速度調整**: 0.5x〜2.0x の範囲で調整可能
- **手動モード**: ドラッグ&ドロップで自由に操作

### 📈 リアルタイム状態表示

- 変数モニター（i, j, k, target, max, min など）
- 一時保存領域（hozon）の可視化
- 発見位置（foundAt）の表示

## 📖 使い方

### 基本操作

1. **アルゴリズムを選択**: ドロップダウンから実行したいアルゴリズムを選択
2. **ターゲット値を設定**: 探索アルゴリズムの場合、探したい値を入力
3. **実行開始**:
   - ▶️ ボタンで自動再生
   - ⏭️ ボタンでステップ送り
   - ⏮️ ボタンでステップ戻し
4. **コードを確認**: Macro/Python タブを切り替えて、異なる言語での実装を確認

### 手動モード

「手動モード」を選択すると、カードを自由にドラッグ&ドロップして操作できます。アルゴリズムの動作を自分で試してみましょう！

### コンテナの役割

| コンテナ | 説明 | 使用アルゴリズム |
|---------|------|-----------------|
| **hozon** | 一時保存領域 | 挿入ソート |
| **foundAt** | 発見位置 | 線形探索、二分探索 |
| **target** | 探索対象の値 | 線形探索、二分探索 |
| **max** | 最大値 | 最大・最小 |
| **min** | 最小値 | 最大・最小 |

## 🎓 対応アルゴリズム

### 線形探索 (Linear Search)

配列を先頭から順番に探索し、目的の値を見つけるアルゴリズム。

**計算量**: O(n)

### 二分探索 (Binary Search)

ソート済み配列を半分ずつ絞り込んで探索するアルゴリズム。

**計算量**: O(log n)

### バブルソート (Bubble Sort)

隣接する要素を比較し、大きい方を右に移動させるソートアルゴリズム。

**計算量**: O(n²)

### 選択ソート (Selection Sort)

未ソート部分から最小値を選択し、先頭に配置するソートアルゴリズム。

**計算量**: O(n²)

### 挿入ソート (Insertion Sort)

未ソート部分の要素を、ソート済み部分の適切な位置に挿入するアルゴリズム。

**計算量**: O(n²)

### 最大・最小 (Min/Max)

配列内の最大値と最小値を同時に探索するアルゴリズム。

**計算量**: O(n)
